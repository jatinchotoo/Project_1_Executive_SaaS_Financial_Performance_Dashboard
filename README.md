## Overview
This project is a Power BI executive dashboard designed to present key
financial performance indicators in a clear, decision-focused format.

The dashboard is structured for senior management review and demonstrates
how financial data can be translated into actionable insights using clean
visuals and consistent design.

## Dashboard Content
The dashboard includes:
- Revenue and profitability overview
- Key financial ratios (ROE, Debt-to-Equity)
- Comparative performance analysis
- Risk indicators using traffic-light logic
- Executive summary visuals suitable for board reporting

## Tools Used
- Power BI Desktop
- DAX measures
- Financial analysis and reporting concepts

## Purpose
This project was created as a portfolio demonstration to showcase financial
analytics, dashboard design, and executive reporting capability.

## Data Disclaimer
All data used in this dashboard is simulated or anonymised and does not
represent any real organisation.
